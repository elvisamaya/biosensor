# Hardware Notes

## Board

Arduino Uno

## Components

- Force-sensitive resistor (FSR)
- Potentiometer
- Tilt switch
- Breadboard
- Jumper wires
- USB cable

## Pin Mapping

- `A0`: force-sensitive resistor
- `A1`: potentiometer
- `D2`: tilt switch

## Setup Summary

The Arduino Uno collects:
- pressure input from the force-sensitive resistor
- analog input from the potentiometer
- motion state from the tilt switch
