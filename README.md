# Biometric Sensing System

This project is an Arduino Uno-based prototype for monitoring simple physiological proxy signals in real time. It uses a force-sensitive resistor, a potentiometer, and a tilt switch to collect sensor input and establish a baseline for future anomaly detection and responsive feedback.

## Hardware Used

- Arduino Uno
- Force-sensitive resistor (FSR)
- Potentiometer
- Tilt switch
- Breadboard
- Jumper wires
- USB cable for Arduino Uno

## How to Run

1. Open `src/biometric_sensing_system.ino` in the Arduino IDE.
2. Connect the Arduino Uno with the USB cable.
3. Select the correct board and port.
4. Upload the code.
5. Open Serial Monitor at `115200` baud.
